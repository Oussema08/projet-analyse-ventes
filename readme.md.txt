# 📊 Analyse des ventes retail

## 🎯 Objectif du projet

Ce projet analyse un dataset de ventes retail (magasin) pour :

- Calculer le chiffre d'affaires total
- Identifier les catégories de produits les plus vendues (quantité et valeur)
- Visualiser l'évolution des ventes dans le temps
- Analyser les tendances par mois et par catégorie

## 📁 Fichiers inclus

| Fichier | Description |
|---------|-------------|
| `analyse_ventes.ipynb` | Notebook Jupyter contenant toute l'analyse |
| `retail_sales_dataset.csv` | Dataset des ventes (1000 transactions) |
| `requirements.txt` | Dépendances Python nécessaires |
| `README.md` | Documentation du projet |
